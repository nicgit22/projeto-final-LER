# HealthManager (Triagem Hospitalar)

 Briefing: "No meu hospital, a ordem de chegada não é tudo. Quero implementar o Protocolo de Manchester. O paciente chega e a enfermeira dá uma cor: Vermelho (Emergência), Amarelo (Urgente) ou Verde (Pouco Urgente). Pacientes com mais de 60 anos sempre passam na frente do Verde, mesmo que tenham chegado depois."

## Principais Desafios de Requisitos:
RN: Lógica de fila por prioridade de cor e idade.

RF: Listar a fila de espera atualizada a cada nova triagem.

RNF: O sistema deve ser rápido para não travar na recepção.
